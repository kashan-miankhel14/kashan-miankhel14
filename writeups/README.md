## Methodology Reference

Each writeup follows a structured approach:
1. **Reconnaissance** — Port scanning, service enumeration, technology identification
2. **Enumeration** — Directory brute-forcing, hidden content discovery
3. **Exploitation** — Leveraging identified vulnerabilities
4. **Privilege Escalation** — Gaining root/admin access
5. **Documentation** — Clear, reproducible steps with tool output
